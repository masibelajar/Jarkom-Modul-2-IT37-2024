2.Karena para pasukan membutuhkan koordinasi untuk melancarkan serangannya, maka buatlah sebuah domain yang mengarah ke Solok dengan alamat sudarsana.xxxx.com dengan alias www.sudarsana.xxxx.com, dimana xxxx merupakan kode kelompok. Contoh: sudarsana.it01.com.

3.Para pasukan juga perlu mengetahui mana titik yang akan diserang, sehingga dibutuhkan domain lain yaitu pasopati.xxxx.com dengan alias www.pasopati.xxxx.com yang mengarah ke Kotalingga.

4.Markas pusat meminta dibuatnya domain khusus untuk menaruh informasi persenjataan dan suplai yang tersebar. Informasi dan suplai meme terbaru tersebut mengarah ke Tanjungkulai dan domain yang ingin digunakan adalah rujapala.xxxx.com dengan alias www.rujapala.xxxx.com.

buka nano /etc/bind/named.conf.local
lalu konfigurasi ini:
di solok
zone "sudarsana.it37.com" {
	type master;
	file "/etc/bind/jarkom/sudarsana.it37.com";
};

dikotalingga
zone "pasopati.it37.com" {
	type master;
	file "/etc/bind/jarkom/pasopati.it37.com";
}; 


di tanjungkulai
zone "rujapala.it37.com" {
	type master;
	file "/etc/bind/jarkom/rujapala.it37.com";
}; 

buka nano /etc/bind/jarkom/it37.com
lalu tambahkan konfigurasi seperti ini:
pada  solok
;
; BIND data file for local loopback interface
;
$TTL    604800
@       IN      SOA     sudarsana.it37.com.  sudarsana.it37.com. (
                          2          ; Serial
                      604800         ; Refresh
                       86400         ; Retry
                     2419200         ; Expire
                      604800 )       ; Negative Cache TTL
;
@       IN      NS      sudarsana.it37.com.
@       IN      A       192.235.2.2       ; IP Solok
@ 	IN      CNAME   sudarsana.it37.com.

pada kotalingga
;
; BIND data file for local loopback interface
;
$TTL    604800
@       IN      SOA     pasopati.it37.com.   pasopati.it37.com. (
                          2          ; Serial
                      604800         ; Refresh
                       86400         ; Retry
                     2419200         ; Expire
                      604800 )       ; Negative Cache TTL
;
@       IN      NS      pasopati.it37.com.
@       IN      A       192.235.1.6       ; IP Kotalingga
@	IN      CNAME   pasopati.it37.com.

pada tanjung kulai
;
; BIND data file for local loopback interface
;
$TTL    604800
@       IN      SOA     rujapala.it37.com.   rujapala.it37.com. (
                          2          ; Serial
                      604800         ; Refresh
                       86400         ; Retry
                     2419200         ; Expire
                      604800 )       ; Negative Cache TTL
;
@       IN      NS      rujapala.it37.com.
@       IN      A       192.235.1.4       ; IP Tanjungkulai
@	IN      CNAME   rujapala.it37.com.
