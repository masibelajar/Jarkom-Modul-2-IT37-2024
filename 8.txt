8.Kamu juga diperintahkan untuk membuat subdomain khusus melacak kekuatan tersembunyi di Ohio dengan subdomain cakra.sudarsana.xxxx.com yang mengarah ke Bedahulu.

pada sriwijaya menambahkan config di /etc/bind/jarkom/sudarsana.it37.com

;
; BIND data file for local loopback interface
;
$TTL    604800
@       IN      SOA     sudarsana.it37.com.  sudarsana.it37.com. (
                          2          ; Serial
                      604800         ; Refresh
                       86400         ; Retry
                     2419200         ; Expire
                      604800 )       ; Negative Cache TTL
;
@       IN      NS      sudarsana.it37.com.
@       IN      A       192.235.2.2       ; IP Solok
@ 	IN      CNAME   sudarsana.it37.com.
cakra   IN      A       192.235.1.5       ; IP Bedahulu
